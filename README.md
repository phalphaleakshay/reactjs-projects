## reactjs-projects
This repository contains SPA web projects with the help of react.js library. 

### Pre-requisite for reactjs
  
  1. NodeJS and NPM
  2. React and React DOM
  3. Webpack
  4. Babel

### Installation 
 
 ``` npx creact-react-app <project_name> ```

### Editor
You can use any of :
Microsoft VS Code, Atom, Bracket.
